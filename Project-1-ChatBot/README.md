# Project 1 - Simple AI ChatBot

## Overview

This project is a simple AI-based chatbot developed using Python.
The chatbot is designed to interact with users through text-based conversations and generate predefined responses based on user input.

This project was developed as part of the DecodeLabs Artificial Intelligence Internship Program to understand the basics of chatbot development, user interaction handling, and Python programming concepts.

---

## Objectives

* Understand the fundamentals of chatbot development
* Learn how user input and responses work
* Implement basic conversational AI logic
* Practice Python programming and debugging

---

## Technologies Used

* Python
* File Handling
* Conditional Statements
* Loops
* Functions

---

## Features

* Interactive text-based conversation
* User-friendly chatbot responses
* Continuous chat interaction loop
* Exit command functionality
* Chat saving/file handling support

---

## Project Workflow

1. User enters a message
2. Chatbot reads the input
3. Program checks matching conditions
4. Bot generates a response
5. Conversation continues until exit command is given
