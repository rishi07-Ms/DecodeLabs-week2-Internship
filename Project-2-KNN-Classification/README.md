KNN Classification using Machine Learning

This project demonstrates the implementation of the K-Nearest Neighbors (KNN) classification algorithm using Python and the Iris dataset from Scikit-learn.

🚀 Features
Data Preprocessing
Feature Scaling
Train-Test Split
KNN Model Training
Accuracy Evaluation
Error Rate Analysis
Confusion Matrix Visualization
Error Rate vs K Value Graph
🛠️ Technologies Used
Python
NumPy
Pandas
Matplotlib
Scikit-learn
📂 Dataset

This project uses the built-in Iris dataset from Scikit-learn.

📊 Output
Accuracy Score
Classification Report
Confusion Matrix
Error Rate Graph
Sample Output

K = 13 | Error Rate = 0.0333
K = 14 | Error Rate = 0.0333
K = 15 | Error Rate = 0.0333

Model Accuracy: 96%+

▶️ How to Run

Install required libraries:

pip install numpy pandas matplotlib scikit-learn

Run the project:

python knn.py
